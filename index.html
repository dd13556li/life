設定

安裝 App

將「人生大哉問」加入主畫面，像原生 App 一樣全螢幕使用、離線閱讀。

顯示大小

調整整體畫面縮放。設定後會記住，下次開啟自動套用（手機與電腦各自獨立記憶）。

AI 功能

輸入 Claude API 金鑰以啟用 AI 生成答案。需搭配本地伺服器使用。

啟動本地伺服器：

python server.py

然後至 http://localhost:8080 開啟本應用

資料管理

匯入的卡片將合併到現有資料中（已存在的 ID 不會被覆蓋）。

個人筆記包含反思日誌、實踐打卡與閱讀記錄，以 Markdown 格式匯出。個人資料備份包含日誌、閱讀歷史與 SRS 資料。

Google Drive 雲端備份

將日誌、閱讀記錄與 SRS 資料備份至 Google Drive，無需帳號系統。
需先至 Google Cloud Console 建立 OAuth 用戶端 ID，並將 http://localhost:8080 加入「已授權的 JavaScript 來源」。

每日複習提醒

開啟後，每次進入 App 時若有待複習卡片，將顯示瀏覽器通知（每天至多一次）。
